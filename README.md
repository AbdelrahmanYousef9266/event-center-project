# Event Center Website

this project is a multi-page website for an event center built using HTML and CSS.

it showcases events,and allows users to send booking inquiries.



## Project Overview

The Event Center website is designed to simulate a real-world event ;
it includes multiple pages, a responsive layout, and interactive elements.


## Features 

-Multi-page website:

-Home page
-Contact page
-Events Page

-navigation menu on all pages

-Event listings and schedule table

-Contact form for booking inquiries

-Footer with social media links

-Responsive design 

## Technologies Used


-HTML5
-CSS3(Flexbox&Grid)
-Responsive Design (Media Queries)
